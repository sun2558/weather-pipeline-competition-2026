# 天擎故事：从电池清洗工到气象系统架构师

## 一句话概括
一个在三星工厂清洗电池的学生，因观察夏季副高异常而决心建造气象数据分析系统，最终开发出“天擎”——一个自动化气象数据处理与可视化平台。

## 关键节点
- **2025.4-8**：天津武清三星视界工厂，清洗18650电池载体，同时追踪副高异常
- **2025.9**：回校启动天擎项目，目标：自动化气象数据分析
- **2026.1**：完成校赛版，具备数据加载→清洗→分析→报告全流程

## 系统能力
- 自动化数据质量检测（3σ原则）
- 专业气象可视化（500hPa位势高度场）
- 可解释报告生成（不仅报异常，还解释可能原因）
